Git: Sistem Kontrol Versi Tersebar

Git adalah sistem kontrol versi yang sangat populer yang digunakan oleh pengembang perangkat lunak untuk mengelola kode sumber proyek mereka. Berikut ini adalah rangkuman singkat mengenai Git:

Sistem Kontrol Versi (Version Control System - VCS)

Git adalah jenis VCS yang memungkinkan Anda melacak perubahan dalam kode sumber proyek perangkat lunak Anda. Ini memungkinkan Anda untuk bekerja dengan tim atau secara individu tanpa kehilangan riwayat perubahan.

Distribusi dan Tersebar

Git adalah sistem kontrol versi terdistribusi, yang berarti setiap anggota tim memiliki salinan penuh dari repositori. Ini memungkinkan kerja kolaboratif tanpa koneksi internet yang konstan.

Repositori

Repositori adalah tempat penyimpanan untuk proyek Git. Ada dua jenis repositori: lokal (di komputer Anda) dan repositori jarak jauh (remote) yang dapat digunakan untuk berkolaborasi dengan orang lain.
